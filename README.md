## Install

```sh
$ npm i react-scrollspy
```


## Usage

```js
import Scrollspy from 'react-scrollspy'

...

<div>

  <div>
    <section id="section-1">section 1</section>
    <section id="section-2">section 2</section>
    <section id="section-3">section 3</section>
  </div>

  <Scrollspy items={ ['section-1', 'section-2', 'section-3'] } currentClassName="is-current">
    <li><a href="#section-1">section 1</a></li>
    <li><a href="#section-2">section 2</a></li>
    <li><a href="#section-3">section 3</a></li>
  </Scrollspy>

</div>
```


## Props

### `items={ Array }`

Id list of target contents.

### `currentClassName={ String }`

Class name that apply to the navigation element paired with the content element in viewport.

### `scrolledPastClassName={ String }`

Class name that apply to the navigation elements that have been scrolled past [optional].

### `componentTag={ String }`

HTML tag for Scrollspy component if you want to use other than `ul` [optional].

### `style={ Object }`

Style attribute to be passed to the generated &lt;ul/&gt; element [optional].

### `offset={ Number }`

Offset value that adjusts to determine the elements are in the viewport [optional].

### `rootEl={ String }`

Name of the element of scrollable container that can be used with querySelector [optional].

### `onUpdate={ Function }`

Function to be executed when the active item has been updated [Optional].

### `highlightAllItemsInView= { Boolean }`

Enables highlighting of all navigation elements that are currently visible in the viewport [Optional].

## Methods

### `offEvent`

Remove event listener of scrollspy.

### `onEvent`

Add event listener of scrollspy.
